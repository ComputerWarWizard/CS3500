Javascript

Standards

ECMA

ECMA owns the JavaScript standardization and has published the standard ECMA-252. The current standard version is version 6.

Tutorials

W3Schools

W3 Schools has a very good tutorial on JavaScript. It lays out the language from the basics to the most advanced features with plenty of examples. It also provides a code simulator which enables you to test what you have learned at the end of each section.

Codecademy

Codecademy is a great resource for the beginner learning JavaScript. After making an account the user is able to jump right into JavaScript. The main focus here is for the user to learn by doing rather than from a book. After a quick paragraph the user can then start to play with what they and learned and see the results instantly.

Examples & Help

W3Schools Examples

W3Schools' big list of javascript examples. Almost anything you'd want to do in javascript can be seen in an example on this website. On top of having many examples, this site also explains the javascript behind what is happening. This is a good site to look at if you're new to javascript. Looking through all the examples can help give you an idea of how to use javascript and waht it can do.

StackOverflow

Stackoverflow is the most common forum on the internet for programming problems. Since it is so common, a lot of people are members which means there is a large body of people who can help solve problems.

CodingForums

Coding forums is one of the top results when searching for an internet forum. It provides expert help on issues that do not seem to be addressed anywhere else.

Advanced Frameworks

W3Schools

W3Schools list of the 3 most popular javascript frameWorks and a brief description of what a library/framework is. Contains information on jQuery, Prototype, and MooTools. This is really just a short introduction to frameworks and should give you an idea of what they are for.

Google Hosted Libraries

A list of google hosted libraries. Google hosts the most popular open source libraries, making them reliably available. There are many available frameworks to choose from. If you already have a good idea of what frameworks are and are just looking to familiarize yourself with some of the most popular ones, this is a good place to look.

Compatibility

GitHub

This chart shows the major differences in JavaScript between browsers. It is important to know what is supported and what is not supported before a webpage is created.

Security

Cross-site Scripting

One of the most widespread issues with javascript security is Cross-site scripting. This wikipedia article is a good resource for reading up on cross-site scripting. This is probably the most complete resource for information on cross-site scripting. It's good for information on cross-site scripting but you will have to look elsewhere for other security issues.

StackExchange

A Q&A resource for all information security. Searching through the tags for javascript reveals many javascript related security questions. If you're unable to find information about a specific security question, this is a good place to ask. There's no garantee that this site will keep you up to date with all the latest security issues, but it's a good spot to watch out for.

Veracode

This sis a small write up on common javascript security issues. It's not a resource you'll constantly refer to for information, but it's a good thing to read through once. It covers Cross-site Scripting and cross-site request forgery.

Tools

List of development tools

A big list of tools for web development. Contains frameworks and libraries as well as editors and IDEs. Not only is it a resource for javascript, but also css and html.

If you're familair with eclipse these tools will make eclipse fit for javascript development. Many people are comfortable with eclipse and this eclipse plugin is for them.

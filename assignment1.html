HTML 5 test

Open this website in any browser to quickly tell what is and isn't compatable. Extensive lists seperated into distinct catagories makes this a handy tool for finding every detail about many browsers. Click on compare to similtaniously learn about five browsers at once. It supports almost every version of every current browser on desktops, mobile, and tablets. It also contains a list of recent site visits showing the browsers score, version, and operating system.

W3Schools provides a list of many common css3 features and shows which browsers support them. It shows all the major browsers and the earliest version that supported the specific feature.
